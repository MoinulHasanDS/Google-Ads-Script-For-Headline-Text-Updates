# Google Ads Responsive Search Ad Year Update Script

This Google Ads Script automatically updates **Responsive Search Ad (RSA) headlines** by replacing an old year (e.g. `2025`) with a new year (e.g. `2026`) across all **enabled RSAs**, while preserving pinned headlines and logging all changes in a Google Sheet.
